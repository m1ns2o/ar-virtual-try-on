# MakeHuman Clothing Assets

The Unity desktop MVP uses selected MakeHuman Community clothing assets.

Source packs:

- `shirts01_cc0.zip`
- `pants01_cc0.zip`
- `dress01_cc0.zip` (`https://static.makehumancommunity.org/assets/assetpacks/dress01.html`)
- `westernized_quipoa` (`http://www.makehumancommunity.org/node/1526`)

Selected demo garments:

- `namuhekam_male_polo_shirt` by namuhekam, CC0
- `toigo_fisherman_sweater` by MargaretToigo, CC0
- `toigo_wool_pants` by MargaretToigo, CC0
- `aethelraed_flapper_dress` by Aethelraed_Unraed, CC0, used as a short-sleeve `OnePiece` garment
- `westernized_quipoa` by Elvaerwyn, CC-BY, used as a high-neck short-sleeve `OnePiece` garment

The original pack metadata is preserved in:

- `unity-ar-closet/ExternalAssets/MakeHuman/shirts01_cc0/packs/shirts01.json`
- `unity-ar-closet/ExternalAssets/MakeHuman/pants01_cc0/packs/pants01.json`
- `unity-ar-closet/ExternalAssets/MakeHuman/dress01_cc0/packs/dress01.json`
- `Assets/ARCloset/MakeHuman/ShortSleeveQipao/westernized_quipoa.mhclo`
- `Assets/ARCloset/MakeHuman/ShortSleeveQipao/westernized_quipoa.mhmat`

The selected garments are CC0 except for `westernized_quipoa`. The `westernized_quipoa` garment is
CC-BY and must retain attribution to Elvaerwyn when redistributed or shown in a
published build. The shift dress is also exposed in the web catalog; its source
pack and attribution are recorded in the Unity catalog asset.
